Declaration of Arrays

<p>Method 1:</p>

```Java
// elementType[] arrayName = {initialValue:index0 .. initialValue:indexN-1};

// example
int[] array = {1, 2, 3, 4, 5};

// All values in the array should be of the same type
```

<p>Method 2:</p>

```Java
// elementType[] arrayName = new elementType[index];
// example
int[] array = new int[5];
```
